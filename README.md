# Customer Portal POC

## Quick Start
1. Backend: 
    - cd backend
    - cp .env.example .env
    # Edit .env and add your ServiceM8 API token
    - npm install
    - npm run dev

2. Frontend: 
    - cd frontend  
    - cp env.local.example env.local
    - npm install
    - npm run dev

See TECH_NOTES.md for detailed technical documentation.
